# Navigation

SwiftUI の代表的な画面遷移を確認するためのサンプルアプリです。

## Included Examples

- `TabView` によるタブ切り替え
- `NavigationStack` と `NavigationLink` による階層遷移
- `.sheet` によるモーダル表示
- `dismiss` によるモーダルの終了

## Tech Stack

- Swift
- SwiftUI

## Getting Started

`Navigation.xcodeproj` を Xcode で開き、`Navigation` scheme を実行してください。

## Purpose

このリポジトリは、SwiftUI のナビゲーションAPIを学習・検証するための小規模なサンプルです。
